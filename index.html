  1. ここでAIに抽出させたい「項目」とその「ルール」を設定し、フォーマットを生成します。
  2. 「生成されたフォーマット ＋ 企業名」をコピーし、お好きなAI（ChatGPTなど）に投げます。
  3. AIが出力した「表（Markdown）」を下のインポート枠に貼り付けます。
  4. 自動的に下の「企業リスト」にデータが追加・管理されます。

抽出したい項目を追加し、それぞれの属性（タグ化やカスタム変数など）を設定してください。

使い方ガイド：AI抽出フォーマットの作成例

以下のフォーマット例は、ガイド等に掲載するための「就活情報抽出用フォーマット」のサンプルです。

📝 AI抽出フォーマット（プロンプト生成内容）例

以下の企業情報を調査・整理し、指定のテキストフォーマットで出力してください。見やすい表形式などのリッチテキスト装飾は【厳禁】です。

■ 抽出項目とルール
- 企業名
- 業界
  (ルール: 業界名には「#IT」などのようにハッシュタグを付けてください)
- 職種
  (ルール: 職種名にはハッシュタグを付けてください)
- 初任給（万円）
  (ルール: 必ず数値のみを抽出し、セルの末尾に <!-- var_salary: (抽出した数値) --> と記載してください)
- 志望度
  (ルール: ★の数で表現し、セルの末尾に <!-- var_star: (数値) --> と記載してください)
  (ルール: 志望度が★4以上の場合は、セルの末尾に <!-- color:green --> と記載してください)
  (ルール: 志望度が★2以下の場合は、セルの末尾に <!-- color:red --> と記載してください)
- 選考ステップ
- インターン締切日
  (ルール: 必ずYYYY-MM-DDの形式にして、セルの末尾に <!-- var_deadline: YYYY-MM-DD --> と記載してください)

このルールを適用することで、AIは以下のような「スラッシュ区切り」の生テキストのみを出力し、コピー＆ペーストで一発で当アプリに取り込めるようになります。

/ 企業名 / 業界 / 職種 / 初任給（万円） / 志望度 / 選考ステップ / インターン締切日 /
/ --- / --- / --- / --- / --- / --- / --- /
/ 株式会社サンプル / #IT / #エンジニア / 25 <!-- var_salary: 25 --> / ★★★★ <!-- var_star: 4 --><!-- color:green --> / ES -> 1次面接 -> 最終面接 / 12月20日 <!-- var_deadline: 2026-12-20 --> /
